(static) new(DomElementopt, positionopt)

Source:
  • libcursor-src.js, line 87
Static method to initialize and return an appropriate kind of cursor.
Example
Cursor.new()  // Returns current cursor under currently focused element.

// Only DomElement specified: returns cursor at the end of the text.
Cursor.new(document.getElementById('mydiv'))
// #mydiv
// +-----------------+
// |           +     |
// |123456789..|     |
// |           +     |
// +-----------------+

// With position specified:
Cursor.new(document.getElementById('mydiv'), 5)
// #mydiv
// +-----------------+
// |     +           |
// |12345|6789..     |
// |     +           |
// +-----------------+
Parameters:
Name Type Attributes Description
DomElement <optional>
 This must be either an element with contenteditable set to true, or a textarea.
position integer <optional>
 This specifies the position of the cursor in the DomElement.
Returns:
An appropriately initialized Cursor object.

deleteBackward(amount)

Source:
  • libcursor-src.js, line 339
Deletes the specified number of characters before the cursor.
Example
//         +
// this is |awesome
//         +
cursor.deleteBackward(3)
//         +                                      +
// this is |awesome -> deleteBackward (3) -> this |awesome
//         +                                      +

//         +---+
// this is |awe|some
//         +---+
cursor.deleteBackward(4)
//     +
// this|awesome
//     +
Parameters:
Name Type Description
amount integer The number of characters to be deleted.

getText(start, end) → {string}

Source:
  • libcursor-src.js, line 366
Get the text surrounding the cursor. Note: The cursor itself does not move.
Example
//         +      
// this is |awesome
//         +     
cursor.getText(-3, 3) === "is awe"
cursor.getText(-4, 0) === " is "
cursor.getText(2, 4) === "es"

//         +---+
// this is |awe|some
//         +---+
cursor.getText(-3, 3) === "is awesom"
cursor.getText(-4, 0) === " is awe"
cursor.getText(2, 4) === "esome"
Parameters:
Name Type Description
start integer Start of the cursor moves this much forward.
end integer End of the cursor moves this much forward.
Returns:
The text enclosed between the start and end of the cursor after the (imaginary) moving is done.
Type
string

insertAfter(text)

Source:
  • libcursor-src.js, line 197
Inserts text at the position of the cursor, keeps the cursor at the start of the inserted text.
Example
//     +
// big | apple
//     +
cursor.insertAfter('rotten')
//     +
// big |rotten apple
//     +

//      +-----+
// this |isn't| good
//      +-----+
cursor.insertAfter('is')
//      +  
// this |is good
//      +
Parameters:
Name Type Description
text string The text to be inserted.
