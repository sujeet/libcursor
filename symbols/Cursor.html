Class Detail
Cursor(DomElement, position)
Represents a Cursor.
Parameters:
DomElement Optional
This must be either an element with contenteditable set to true, or a textarea or an input field. If unspecified, this would return a cursor at the current visible position of the cursor on the page.
position Optional
This specifies the position of the cursor in the DomElement. When unspecified, while the DomElement is specified, the resultant cursor will be at the start of the editing area. In case of unspecified DomElement, the cursor at the current visible cursor position will be returned.
